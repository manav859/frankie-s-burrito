# Image And Media Optimization

Use this for images, video, and other visual assets.

Workflow:
1. Determine intrinsic size, rendered size, and priority.
2. Serve appropriately sized assets for each viewport.
3. Preserve aspect ratio to prevent layout shift.
4. Lazy-load non-critical media and preload only what matters.
5. Keep accessible alt text and captions aligned with editorial content.

Rules:
- Large original uploads are not a delivery strategy.
- Decorative media should not compete with meaningful content.
- Media handling should fail gracefully when CMS assets are missing or malformed.
