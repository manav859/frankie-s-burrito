# On-Page SEO Structure

Use this when building or reviewing page templates and content sections.

Workflow:
1. Confirm one primary topic per page.
2. Keep title, description, H1, and body intent aligned but not duplicated mechanically.
3. Use semantic headings in a clean outline.
4. Support internal links, descriptive anchors, and crawlable navigation.
5. Validate CMS fields so editors can maintain the structure without code changes.

Rules:
- Headings should communicate structure, not just styling.
- Avoid thin template sections that create noise without search value.
- Render metadata and visible copy from compatible content models.
